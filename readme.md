# 设计模式总结－java

## note

- 编译环境 Linux java 1.8.0_144
- 每个子目录下有编译与执行脚本`run.sh`


## reference

- 本文代码绝大部分参考于 [design-pattern-java-source-code]，在此基础上写下自己的理解与笔记，在代码结构上，有自己的Linux风格，用命令行来执行与打包，而不是用ide的傻瓜式操作
